
## Mozzarella (Front - End)

#### Webpack assembly

## Requirements 

**Node.js:** v16+

## Let`s start 

**Install modules:** 

```bash
  npm i --force
```

**To Dev server start:** 

```bash
  npm run dev
```

**Create production build:** 

```bash
  npm run build
```

## Production folder


```bash
  /dist
```
